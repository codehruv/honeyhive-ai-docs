---
title: "Create Dataset"
api: "POST https://honeyhive.ai:5000/datasets"
description: "This endpoint creates a new dataset."
---

### Body

<ParamField body="name" type="string">
  The name of the dataset.
</ParamField>

<ParamField body="purpose" type="string">
  The purpose of the dataset.
</ParamField>

<ParamField body="file" type="object">
  The file containing the dataset.
</ParamField>

<ParamField body="task" type="string">
  The task associated with the dataset.
</ParamField>

<ParamField body="description" type="string">
  The description of the dataset.
</ParamField>

### Response

<ResponseField name="data" type="object">

The contents of the dataset
