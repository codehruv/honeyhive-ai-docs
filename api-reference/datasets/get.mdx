---
title: "Get Datasets"
api: "GET https://honeyhive.ai:5000/datasets"
description: "This endpoint gets the datasets associated with a task."
---

### Body

<ParamField body="dataset_id" type="string">
  Optional parameter to fetch a particular dataset by id.
</ParamField>

<ParamField body="task" type="string">
  Optional parameter to fetch datasets associated with a particular task.
</ParamField>

<ParamField body="prompt" type="string">
  Optional parameter to fetch datasets associated with a particular prompt.
</ParamField>

<ParamField body="purpose" type="string">
  Optional parameter to fetch datasets associated with a particular purpose.
</ParamField>

### Response

<ResponseField name="data" type="object">

The contents of the dataset list

<Expandable title="Toggle object">

<ResponseField name="object" type="string">
  This is the data object associated with the dataset.
</ResponseField>

<ResponseField name="bytes" type="string">
  This is the size of the dataset in bytes.
</ResponseField>

<ResponseField name="filename" type="string">
  This is the filename of the dataset.
</ResponseField>

<ResponseField name="purpose" type="string">
  This is the purpose of the dataset. Possible values are 'Training', 'Validation' and 'Test'.
</ResponseField>

<ResponseField name="description" type="string">
  This is the description of the dataset.
</ResponseField>

<ResponseField name="created_at" type="string">
  This is the date and time when the dataset was created.
</ResponseField>

<ResponseField name="id" type="string">
  This is the unique identifier for the dataset.
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.honeyhive.ai/datasets' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "task": "Email writer"
}'
```

</RequestExample>
