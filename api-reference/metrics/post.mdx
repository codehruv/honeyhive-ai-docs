---
title: "Post Metrics"
api: "POST https://honeyhive.ai:5000/metrics"
description: "This endpoint adds a metric to the task."
---

### Body

<ParamField body="task" type="string">
  Optional parameter to specify the task name.
</ParamField>

<ParamField body="name" type="string">
  Name of the metric.
</ParamField>

<ParamField body="code_snippet" type="string">
  Code snippet of the metric.
</ParamField>

### Response

<ResponseField name="data" type="object">

The contents of the metric

<Expandable title="Toggle object">

<ResponseField name="task_name" type="string">
  This is the name of the project linked to this metric.
</ResponseField>

<ResponseField name="name" type="string">
  This is the name of the metric.
</ResponseField>

<ResponseField name="description" type="string">
  This is the description of the metric - ideally what it does.
</ResponseField>

<ResponseField name="code_snippet" type="string">
  This is the code snippet of the metric in python. Make sure it returns a float or None.
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://honeyhive.ai:5000/metrics' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "task": "Sandbox - Email Writer",
    "name": "Number of sentences",
    "description": "Count number of sentences in generated text",
    "code_snippet": "def num_sentences(generation):
    return len(generation["generation"].split('.'))

result = num_sentences(generation)"
}'
```

</RequestExample>

    "inserted": true
}
