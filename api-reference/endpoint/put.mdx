---
title: "Change Task"
api: "PUT https://honeyhive.ai:5000/tasks"
description: "This endpoint changes the details of a particular task."
---

### Body

<ParamField body="task_id" type="string">
  The unique identifier of the task to be changed.
</ParamField>

<ParamField body="name" type="string">
  The new name of the task.
</ParamField>

<ParamField body="type" type="string">
  The new type of the task. Possible values are 'Text Generation', 'Code Generation', 'Classification', 'Summarization' and 'Other'.
</ParamField>

<ParamField body="prompts" type="array">
  The new list of prompts saved for the task.
</ParamField>

<ParamField body="fine_tuned_models" type="array">
  The new list of fine-tuned models that have been trained for the task.
</ParamField>

<ParamField body="datasets" type="array">
  The new list of datasets that have been uploaded for the task.
</ParamField>

<ParamField body="metrics" type="array">
  The new list of metrics that have been setup for the task.
</ParamField>

### Response

The contents of the updated task
